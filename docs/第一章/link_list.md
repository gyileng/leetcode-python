# 链表

**定义**  
链表是一种物理存储单元上非连续、非顺序的存储结构，数据元素的逻辑顺序是通过链表中的指针链接次序实现的。链表由一系列结点
（链表中每一个元素称为结点）组成，结点可以在运行时动态生成。
